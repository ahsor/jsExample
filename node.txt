Node.js는 모듈을 이용해 기능을 확장할 수 있다. 
모듈은 기능을 쉽게 사용하고자 메서드와 속성을 미리 정의해 모아 놓은 것이다. 
모듈을 사용하려면 다음과 같이 두개의 js 파일이 필요하다. 
1. experts 사용법
module.js
exports.abs = function (number) {
  if (0 < number) {
    return number;
  } else {
    return -number;
  }
};

exports.circleArea = function (radius) {
  return radius * radius * Math.PI;
};


main.js
let m = require("./module");

console.log(m.abs(-273));
console.log(m.circleArea(20));

실행
node 10_module/main.js
273
1256.6370614359173

* 모듈은 이름만 지정하면 해당 파일을 찾아 불러들이지만 만약 파일이 없다면 
폴더명을 찾아 그안의 파일을 불러들이기도 한다. 
require("./module")지정하면 module.js를 찾아보고 파일이 없는 경우 
module폴더를 찾아 그낭의 index.js를 불러들인다.
